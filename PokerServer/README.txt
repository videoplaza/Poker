Java
--------------------------------

To edit your bot:
1. edit the YourBot class
2. use YourBot in BotServlet instead of ExampleBotImpl

To test your bot:
1. open PokerBotTest
2. replace ExampleBotImpl with your own bot
3. run the test

To start the server:
1. open BotService
2. run main


Python
--------------------------------

To use python bot:
1. open the file bot.py
2. edit the file
3. replace ExampleBotImpl with JythonBot where applicable
